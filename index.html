Projects


I created an infomrative website about the Gender Wage Gap eating up our workforce and economy. Over the years, I have updated the website from the standard CSS/HTML configuration to also using Bootstrap 5 features.

I implemented parallax scrolling on the "Learn More" page and used a Google Maps API on the "Take Action" page to give an immersive feel to the website and engage the audience.

If you click on the website image, you can explore the website too!

Last semester (Fall 2022) as part of my General Engineering class, I participated in a Semester-Long Design project in a group of two other members. Our project consisted of creating a commercially viable drowsy driving sensor and focused facial recognition software. For hardware, we used a Raspberry Pi 4, a PiCamera, and an Ultrasonic Sensor. For Software, Python was used in the Raspberry Pi IDE and two very important libraries: CV2 and "face_recognition." I was responsible for creating the CAD files for the product and 3D printing it as well. I also contributed to the electric wiring and facial recognition/ultrasonic sensor code.
